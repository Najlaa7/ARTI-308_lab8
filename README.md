# ARTI-308_lab8
In this lab, we applied the K-Nearest Neighbors (KNN) algorithm on a given dataset to perform classification. The process began with exploratory data analysis (EDA) using visualizations such as pairplots. The features were then standardized using StandardScaler to improve model performance.
After splitting the data into training and testing sets, a KNN model was trained and evaluated using a confusion matrix and classification report. Finally, the optimal value of K was determined using the elbow method, and the model was retrained to achieve better accuracy.
